## 二、搭建步骤清单

### 阶段 1：版本控制（Git）

```powershell
cd C:\Users\11576\Desktop\新建文件夹
git init
git add .
git commit -m "init: 项目基础环境骨架"
```

建议创建远程仓库（Gitee / GitHub），邀请组员加入。

### 阶段 2：Docker 基础服务

```powershell
cd docker
docker compose up -d
docker compose ps          # 确认三个容器均为 running/healthy
docker compose logs mysql  # 查看数据库初始化日志
```

### 阶段 3：验证数据库

使用 Navicat / SQLyog 连接：

- 主机：`127.0.0.1`
- 端口：`3306`
- 用户名：`creditbank`
- 密码：`creditbank123`
- 数据库：`credit_bank`

确认以下表已创建：`sys_user`、`credit_account`、`learning_archive` 等。

### 阶段 4：启动后端

```powershell
cd backend
.\mvnw.cmd clean package -DskipTests
.\mvnw.cmd spring-boot:run
```

首次运行会下载 Maven 及依赖，耗时约 5–10 分钟（视网络而定）。

浏览器访问：http://localhost:8080/api/health  
期望返回：`{"code":200,"message":"success","data":{"status":"UP",...}}`

### 阶段 5：启动前端

```powershell
cd frontend
npm install
npm run dev
```

浏览器访问：http://localhost:5173，首页应显示「系统状态检测」为 UP。

### 阶段 6：IDE 配置

**IntelliJ IDEA 2020+**

1. File → Open → 选择 `backend` 目录
2. 等待 Maven 依赖下载完成
3. 设置 Project SDK 为 Java 21
4. Run Configuration：Spring Boot → `CreditBankPlatformApplication`

**VS Code / HBuilderX（前端）**

1. 打开 `frontend` 目录
2. 终端执行 `npm run dev`
